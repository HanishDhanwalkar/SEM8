$S = \{S_{i, i\in N}\}\rightarrow$ set of actions \\
$U = \{U_{i, i\in N}\}\rightarrow$ set of utilities \\
$U_i(a_i, a_{-i}) \forall a_i \in S_i, a_{-i} \in S_{-i} = S_1 \times S_2 \times \dots S_{i+1} \dots$\\

\[ 
U_1(a_1, a_{-1})= \left\{
  \begin{array}{lr}

    x(p_2)/2 &  a_2 = 2
    \end{array}
\right.

    \end{array}
\right.
